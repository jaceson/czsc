### 步骤 2：运行回测

训练完成后，直接运行回测：

```bash
python backtest_xgboost.py
```

**回测脚本会自动：**
- 检测并使用最新模型 (`xgboost_stock_clf_v2.json`)
- 加载对应的特征配置
- 在 2024 年回测区间生成交易信号

### 步骤 3：调整回测参数（可选）

如果回测结果不理想，可调整以下参数：

```python
# backtest_xgboost.py 中的配置
BACKTEST_START = "2024-01-01"   # 回测起始日
BACKTEST_END = "2024-12-31"     # 回测结束日
PROBA_THRESHOLD = 0.55          # 预测概率阈值
USE_TOP_PCT_BY_PROBA = 2        # 只做概率最高的 2% 信号
MAX_SYMBOLS_BACKTEST = 200      # 回测股票数量
```

**调整建议：**
- `USE_TOP_PCT_BY_PROBA = 5` → 更多交易信号（更宽松）
- `USE_TOP_PCT_BY_PROBA = 1` → 更少但更精选的信号（追求高胜率）
- `PROBA_THRESHOLD = 0.50` → 降低阈值增加信号数量

## 🔧 主要优化内容

### 1. 特征工程增强（4 个 → 30+ 个）

#### 趋势型指标
- MACD (DIF, DEA, 柱状图，金叉信号)
- EMA 均线系统 (5, 20, 60 日)
- 价格/均线比率

#### 摆动型指标
- RSI (6, 12, 24 周期)
- KDJ (完整 K, D, J)
- CCI (14, 84 周期)

#### 波动率指标
- BOLL 布林带 (上轨，中轨，下轨，%B, 带宽)

#### 成交量指标
- 成交量比率 (Vol Ratio)
- OBV 能量潮
- VR 成交量比率

#### 动量指标
- ROC 变动速率
- MTM 动量

#### 价格形态
- 价格位置 (Price Position)
- 连续上涨/下跌天数

### 2. 数据处理优化
- **StandardScaler 标准化** - 所有特征标准化处理
- **改进下采样** - 2:1 负正比，保留更多信息
- **防止数据泄露** - 严格时序划分

### 3. 模型训练增强
- **Optuna 超参数调优** - 自动搜索最佳参数组合
- **正则化** - L1/L2 正则化防止过拟合
- **Early Stopping** - 30 轮耐心，自动停止

### 4. 评估指标完善
- Accuracy（准确率）
- AUC（ROC 曲线下面积）
- Precision（精确率）
- Recall（召回率）
- F1-Score（综合指标）
- **特征重要性排序** - 理解哪些指标最重要

## ⚠️ 常见问题

### Q1: "未找到模型或元信息文件"
**解决：** 先运行 `train_xgboost.py` 完成训练

### Q2: "回测区间内无有效交易"
**原因：** 
- 模型太旧，特征不匹配
- PROBA_THRESHOLD 设置过高
- 回测区间数据不足

**解决：**
1. 确认已生成 `xgboost_stock_clf_v2.json`
2. 降低 `PROBA_THRESHOLD` 到 0.50
3. 检查数据是否在 `data/sqlite3.db` 中存在

### Q3: Optuna 安装失败
**可选方案：**
```python
# train_xgboost.py 中设置
USE_OPTUNA = False  # 使用默认参数训练
```

### Q4: 训练时间过长
**优化：**
```python
OPTUNA_TRIALS = 20  # 减少调优次数（默认 50）
MAX_SYMBOLS = 50    # 减少训练股票数量
```
